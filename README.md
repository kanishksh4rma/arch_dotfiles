# arch_dotfiles
Arch Hyprland dotfiles as of Feb 2025

Installation:
```
git clone https://github.com/kanishksh4rma/arch_dotfiles.git
cd config
mv .bashrc ~
mv random_wallpaper.sh ~/Documents
mv * ~/.config
cd ~/Documents
git clone https://github.com/kanishksh4rma/wallpapers.git
mv wallpapers walls
```
